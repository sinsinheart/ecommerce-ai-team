# 恶意订单识别规则库

## 识别模型

### 规则1：高频退货检测
```
条件：
- 买家近30天退货次数 ≥ 3次
- OR 买家历史退货率 > 30%
- OR 同一收货地址关联多个账号频繁退货

动作：自动标记 → 升级人工复核 → 要求举证材料
置信度：中高
```

### 规则2：地址异常检测
```
条件：
- 收货地址为驿站/代收点且退货率异常
- 同一地址关联5个以上不同买家账号
- 收货地址与IP地址跨省且频繁退货

动作：标记可疑 → 不自动通过退款 → 人工审核
置信度：中
```

### 规则3：账号行为异常
```
条件：
- 新注册账号（<30天）立即购买高单价商品
- 账号购买记录仅含退款/售后订单
- 同设备登录多个账号交叉下单

动作：标记风险账号 → 限制自动通过额度
置信度：高
```

### 规则4：订单特征异常
```
条件：
- 同一商品多次下单后退货（调包嫌疑）
- 退货商品与下单商品型号/规格不符
- 退货理由与举证材料不一致
- 退货包裹重量与下单商品不符

动作：拒绝自动退款 → 要求提供完整开箱视频 → 上报平台
置信度：高
```

### 规则5：批量退款攻击
```
条件：
- 短时间内（<1小时）同一商品多次退款申请
- 退款理由高度相似或完全相同
- 下单时间集中在某时间段内

动作：批量冻结 → 统一上报平台 → 保留举证材料
置信度：高（竞对恶意行为）
```

## 买家信誉分层

| 信誉等级 | 判断条件 | 自动通过额度 | 复核阈值 |
|----------|---------|-------------|----------|
| 🌟🌟🌟 优质 | 注册>1年 + 退货率<5% | 500元 | 2000元+ |
| 🌟🌟 正常 | 注册>3月 + 退货率<15% | 200元 | 500元+ |
| 🌟 新用户 | 注册<3月 | 50元 | 100元+ |
| ⚠️ 观察 | 退货率15-30% | 不自动通过 | 全部复核 |
| 🚫 风险 | 退货率>30% 或 有记录 | 拒绝自动通过 | 全部复核+举证 |

## 处理策略矩阵

| 订单金额 | 买家等级 | 策略 |
|---------|---------|------|
| <100元 | 任意 | 自动通过（优先体验） |
| 100-500元 | 优质/正常 | 自动通过 |
| 100-500元 | 新用户/观察 | 人工复核 |
| 100-500元 | 风险 | 人工复核+举证 |
| 500-2000元 | 优质 | 自动通过（信任） |
| 500-2000元 | 正常 | 快速人工复核 |
| 500-2000元 | 其他 | 严格人工复核+视频举证 |
| >2000元 | 任意 | 全部严格人工复核 |

## 防损建议

1. 高价值商品发货前拍照/录像留证
2. 快递签收备注"请当面验货"
3. 退货入库全程录像
4. 定期分析退货数据，发现异常模式
5. 建立买家黑名单共享机制（行业群）
