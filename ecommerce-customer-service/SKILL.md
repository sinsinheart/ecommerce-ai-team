# 智能客服代理

电商智能客服模块，替代人工完成买家日常咨询接待，降低客服人力成本。

## 触发场景

- 买家发送咨询消息（物流、售后、尺码、优惠、退换货等）
- 需要自动应答高频 FAQ
- 检测到潜在差评风险订单，需主动触达安抚
- 多店铺/多平台客服消息统一管理

## 核心工作流

### 1. 消息接收与分类

收到买家消息后，按以下分类路由：

```
Task Progress:
- [ ] 解析消息意图（物流/售后/尺码/优惠/投诉/其他）
- [ ] 查询订单上下文（订单状态、物流节点、商品信息）
- [ ] 判断紧急程度（普通咨询 / 投诉预警 / 差评风险）
- [ ] 匹配应答策略
```

### 2. 高频问题自动应答

- **物流查询**：查询物流API获取最新轨迹，回复预计送达时间
- **售后政策**：返回退换货规则、时效、运费承担规则
- **尺码推荐**：基于商品尺码表 + 用户提供的身高体重，给出推荐尺码
- **优惠券**：告知当前可用优惠、使用条件、领取方式
- **商品咨询**：返回商品规格、材质、使用说明等信息

### 3. 差评风险识别与干预

- 监控订单异常事件：物流延迟、发货超时、商品缺货
- 识别买家负面情绪关键词（不满、投诉、差评、退款）
- 标记高风险订单，主动发送安抚话术和补偿方案
- 自动升级复杂投诉至人工客服

### 4. 应答话术原则

- 称呼买家昵称，语气亲切自然
- 先共情后解决：理解买家情绪，再提供方案
- 给出明确时间节点，避免模糊承诺
- 每次对话结束时确认问题已解决

- 咨询类型、解决状态、耗时

- 买家满意度预估
