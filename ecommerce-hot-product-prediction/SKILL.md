# 全网爆款智能预测与自动上架

爆款预测机器人，实现热点识别→爆款预测→商品优化→自动上架→数据复盘全流程自动化，大幅提升选品效率和店铺流量。

## 触发场景

- 需要全网扫描最新热销商品和趋势
- 需要预测某个品类未来7-30天的潜力爆款
- 需要自动生成新品标题、详情页、主图文案
- 需要一键完成多平台多店铺上架铺货
- 需要复盘爆款预测准确率，迭代选品模型

## 核心工作流

### 1. 全网多渠道信息实时抓取

```
Task Progress:
- [ ] 确定监控品类和平台范围
- [ ] 轮询抓取以下数据源：
  - [ ] 电商平台：销量排行、搜索热度、新品上架趋势、价格分布
  - [ ] 短视频/直播平台：带货视频播放量、商品点击量、达人带货数据
  - [ ] 图文种草社区：种草笔记数量、互动量、提及商品、用户讨论关键词
  - [ ] 热搜/趋势榜单：搜索上升词、热门话题、季节性关键词
  - [ ] 货源平台：新品发布、批发热度、供应链趋势
- [ ] 数据清洗去重，整合为统一格式
```

### 2. AI智能爆款识别与趋势预测

**六大评估维度：**

| 维度 | 权重 | 评估指标 |
|------|------|----------|
| 热度涨幅 | 25% | 搜索量/播放量/笔记数7日增长率 |
| 销量增速 | 25% | 全网销量30日增长趋势 |
| 用户讨论度 | 15% | 评论/分享/收藏等互动数据 |
| 同款店铺数 | 10% | 竞争密度（越少越好） |
| 市场空白度 | 15% | 需求供给比、本店品类覆盖缺口 |
| 季节适配性 | 10% | 是否匹配当前/近期季节需求 |

**爆款分类：**
- 🔥 **短期爆款**（1-7天内爆发）：热点事件驱动、达人带火
- 📈 **长效趋势款**（持续增长30天+）：消费升级、生活方式变化
- 🎯 **季节性爆款**：换季刚需、节日礼品
- 💎 **蓝海潜力款**：低竞争高需求，适合中小商家切入

**预测输出：**
- 潜力爆款清单（含商品类目、预测热销概率、建议定价区间）
- 未来7/15/30天流量走势预测
- 竞争分析（同款店铺数、竞品价格分布）
- 推荐上架时间窗口

### 3. 爆款商品AI精细化优化

- 自动抓取优质同款和TOP竞品详情
- AI生成专属标题（含核心关键词 + 差异化卖点）
- AI生成详情页文案（场景化描述 + 痛点解决 + 信任背书）
- AI生成主图宣传文案（8-15字核心卖点）
- 自动检测违禁词，优化关键词权重
- 结合本店客单价和人群定位差异化优化，避免同质化竞争

### 4. 智能审核与全自动上架

**上架前校验清单：**
```
- [ ] 商品标题合规性检查（违禁词、商标、广告法）
- [ ] 商品类目匹配正确
- [ ] 价格设置在建议范围内
- [ ] 库存信息填写完整
- [ ] 规格参数与实物一致
- [ ] 图片尺寸和数量符合平台要求
- [ ] 运费模板已关联
```

**上架策略：**
- 支持自定义上架时间段（避开平台流量低谷）
- 支持分批铺货（测试款 → 主力款 → 全量款）
- 一键同步多平台、多店铺
- 上架后自动监测新品数据

### 5. 爆款数据复盘迭代

**每日追踪指标：**
- 预测爆款上架后的访客数、转化率、销量
- 对比预测数据 vs 实际表现
- 统计预测准确率（命中率/误报率）

**迭代机制：**
- 分析预测偏差原因（高估/低估/完全未命中）
- 调整六大维度权重
- 更新品类趋势词库
- 生成选品周报供商家参考

**周报模板：**
```markdown
# 选品周报 — YYYY-MM-DD ~ YYYY-MM-DD

## 本周预测爆款复盘
| 商品 | 预测热度 | 实际上架销量 | 准确度 | 分析 |
|------|----------|--------------|--------|------|

## 行业趋势速览
## 下周选品建议
## 模型迭代记录
```
