# 店铺数据指标解读手册

## 核心指标定义

### 流量指标
| 指标 | 计算方式 | 健康值 | 预警线 |
|------|---------|--------|--------|
| 访客数(UV) | 独立访问用户数 | 品类相关 | 环比下降>20% |
| 浏览量(PV) | 页面总浏览次数 | PV/UV=3-5 | PV/UV<2 |
| 跳失率 | 仅访问一页即离开/UV | <50% | >70% |

### 转化指标
| 指标 | 计算方式 | 健康值 | 预警线 |
|------|---------|--------|--------|
| 下单转化率 | 下单UV/访客UV | 品类相关(3-8%) | 低于同行均值30% |
| 支付转化率 | 支付UV/下单UV | >85% | <70%（大量放弃支付） |
| 客单价 | 成交额/支付买家数 | 品类相关 | 连续下降 |
| UV价值 | 成交额/访客UV | 品类相关 | 低于同行均值 |

### 服务指标
| 指标 | 计算方式 | 健康值 | 预警线 |
|------|---------|--------|--------|
| DSR综合 | 商品+服务+物流均值 | >4.7 | <4.5 |
| 商品DSR | 描述相符评分 | >4.7 | <4.5 |
| 服务DSR | 卖家服务态度评分 | >4.7 | <4.5 |
| 物流DSR | 物流服务评分 | >4.7 | <4.5 |
| 退货率 | 退货订单/总订单 | <品类均值 | >品类均值2倍 |
| 纠纷率 | 平台介入纠纷/总订单 | <1% | >2% |

### 商品指标
| 指标 | 说明 | 健康值 |
|------|------|--------|
| 动销率 | 有销量SKU数/总SKU数 | >60% |
| 爆款占比 | TOP5商品销售额/总销售额 | 不超过50%（避免单品依赖） |
| 新品动销率 | 上架30天内新品动销率 | >30% |

## 异常诊断决策树

### 流量下降诊断
```
流量下降 >20%
├── 搜索流量下降为主
│   ├── 关键词排名下滑 → 优化标题+推广
│   └── 搜索热度下降 → 品类趋势变化，考虑换品
├── 推荐流量下降为主
│   ├── 商品权重下降 → 提升转化率+好评率
│   └── 人群标签偏移 → 检查近期推广人群变化
└── 付费流量ROI下降
    ├── 素材疲劳 → 更换创意图
    └── 竞争加剧 → 调整出价策略
```

### 转化率下降诊断
```
转化率下降 >20%
├── 加购率正常但转化低
│   ├── 价格问题 → 对比竞品价格
│   ├── 促销乏力 → 增加优惠手段
│   └── 运费过高 → 考虑包邮或调整运费
├── 加购率也下降
│   ├── 主图/标题不吸引 → 优化内容
│   ├── 评价变差 → 处理差评
│   └── 竞品上同款 → 差异化卖点
└── 流量大幅上涨但转化下降
    └── 来了不精准流量 → 优化关键词精准度
```

## 数据复盘节奏

| 类型 | 频率 | 内容 | 建议推送时间 |
|------|------|------|-------------|
| 日报 | 每日 | 核心指标+异常预警 | 次日9:00 |
| 周报 | 每周一 | 周趋势+竞品+优化建议 | 周一9:30 |
| 月报 | 每月1日 | 月度总结+下月目标+策略调整 | 每月1日10:00 |
| 专项分析 | 按需 | 大促复盘/新品复盘/问题诊断 | 按需 |
