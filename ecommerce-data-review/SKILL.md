# 店铺数据复盘机器人

自动化店铺数据采集、分析与可视化复盘，帮助商家快速掌握经营状况，及时发现经营问题。

## 触发场景

- 每日定时自动生成店铺日报
- 每周/每月自动生成周报/月报
- 商家主动查询店铺实时数据
- 流量或销售额异常波动需要诊断
- 单品数据异常需要专项分析

## 核心工作流

### 1. 数据自动采集

```
Task Progress:
- [ ] 连接店铺后台API，获取以下数据：
  - [ ] 整体指标：访客数、浏览量、成交额、转化率、客单价
  - [ ] 流量来源：搜索/推荐/付费/其他渠道占比
  - [ ] 商品排行：销量TOP10、流量TOP10、滞销商品
  - [ ] 竞品对比：同层级店铺均值参考
- [ ] 对比昨日/上周同期数据，计算环比变化
- [ ] 数据清洗：排除异常刷单、大促期间的特殊波动
```

### 2. 诊断分析

**流量异常诊断：**
- 搜索流量下降 → 关键词排名下滑、竞品抢占
- 推荐流量下降 → 商品权重下降、人群标签偏移
- 付费流量ROI下降 → 出价策略需调整、素材疲劳

**转化率异常诊断：**
- 转化率下滑 → 价格竞争力不足、评价差评增多、详情页信息不足
- 加购率高但转化低 → 价格或运费问题、缺少紧迫感

**单品诊断：**
- 高流量低转化 → 详情页或价格优化
- 高转化低流量 → 提升关键词覆盖和推广
- 持续下滑 → 竞品替代、市场需求变化

### 3. 报告生成

**日报模板：**
```markdown
# 店铺日报 — YYYY-MM-DD

## 核心指标概览
| 指标 | 今日 | 昨日 | 环比 | 同行均值 |
|------|------|------|------|----------|
| 访客数 | | | | |
| 成交额 | | | | |
| 转化率 | | | | |
| 客单价 | | | | |

## 流量来源分布
## TOP5商品
## 异常预警
## 今日优化建议
```

**周报增加：**
- 周趋势图（7天数据曲线描述）
- 单品周表现排名变化
- 竞品动态追踪
- 下周优化重点

### 4. 智能预警

触发预警的条件：
- 单日访客数环比下降 >20%
- 单日成交额环比下降 >30%
- 店铺转化率连续3天低于行业均值
- 主力单品流量连续5天下降
- 出现集中差评或退货
- DSR评分连续下降

预警输出：异常指标、可能原因、建议方案、处理优先级。

## 数据安全

- 店铺数据仅存储于本地工作区
- 报告可自动推送至商家飞书/企业微信
- 不外传任何店铺敏感数据
