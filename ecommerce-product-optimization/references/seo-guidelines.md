# 电商搜索优化指南

## 标题优化原则

### 标题结构公式
```
核心词 + 属性词 + 场景词 + 卖点词

核心词：品类词，如「连衣裙」「充电宝」
属性词：材质/规格/风格，如「纯棉」「20000mAh」「ins风」
场景词：使用场景，如「通勤」「出游」「学生」
卖点词：差异化优势，如「显瘦」「快充」「防水」
```

### 各平台标题字数限制
| 平台 | 限制 | 建议 |
|------|------|------|
| 淘宝 | 30字 | 25-28字最佳 |
| 拼多多 | 30字 | 20-25字，重关键词 |
| 京东 | 50字 | 30-40字，品牌+型号 |
| 抖音 | 36字 | 15-25字，短而精 |
| 快手 | 30字 | 15-20字 |

### 关键词布局策略

**前端高频词（搜索入口）：**
- 使用生意参谋/多多情报通等工具查搜索热词
- 优先选择搜索量大、竞争度中等、与商品高度匹配的词
- 前8-10个字决定搜索匹配度，核心词前置

**中段属性词（精准匹配）：**
- 规格、材质、风格等筛选属性
- 帮助买家在搜索结果中确认"这就是我要的"

**后端长尾词（补量）：**
- 组合词、场景词、人群词
- 覆盖更多长尾搜索需求

### 禁忌
- 堆砌关键词导致不可读 → 降权
- 使用无关热词蹭流量 → 违规下架
- 标题与商品实际不符 → 退款率高
- 频繁修改标题 → 搜索权重重置

## 详情页优化

### 移动端优先原则
- 首屏决定留还是走（前3屏最关键）
- 段落短：每段不超过3行
- 图文混排：文字段落后紧跟演示图
- 字体大小适合手机阅读

### 详情页黄金结构
```
1. 痛点海报（2-3屏）— 激发需求
2. 产品亮点（2-3屏）— 为什么选我
3. 规格参数（1-2屏）— 客观数据
4. 细节展示（3-5屏）— 品质感
5. 使用场景（2-3屏）— 代入感
6. 对比优势（1-2屏）— 和竞品的差距
7. 信任背书（1-2屏）— 认证/质保/评价
8. 行动号召（1屏）   — 立即购买
```

### 详情页常见问题
- 文字堆砌无分段 → 买家直接划走
- 夸大宣传 → 差评风险
- 关键信息缺失（尺寸/材质/保养） → 售后纠纷
- PC端思维做长图 → 手机看不清

## 主图优化

### 主图像素要求
- 淘宝：800×800以上，建议1:1正方形
- 拼多多：800×800以上
- 抖音：1:1或3:4竖图
- 主图视频可提升转化率20-30%

### 主图文案规则
- 第一张主图（搜索展示图）：干净、突出商品主体
- 第二张起可加卖点文案
- 文案不超过15字，字号适中
- 避免使用平台禁用的特殊字符

## 搜索权重影响因素

| 因素 | 权重估计 | 说明 |
|------|---------|------|
| 标题关键词匹配 | 30% | 与买家搜索词的相关性 |
| 销量与转化率 | 25% | 商品的市场表现 |
| DSR评分 | 15% | 商品/服务/物流评分 |
| 评价数量与质量 | 10% | 特别是带图好评 |
| 上下架时间 | 5% | 临近下架有短暂加权 |
| 店铺权重 | 15% | 店铺整体表现和层级 |
