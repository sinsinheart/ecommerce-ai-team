# 广告法违禁词库

## 极限词（绝对禁止）

### 「最」类
最好、最佳、最大、最小、最新、最先进、最高级、最低价、最便宜、最受欢迎、最时尚、最舒适、最有效、最完美、最强、最棒、最牛、最火

### 「一」类
第一、唯一、独一无二、销量第一、排名第一、仅此一家、全国第一、全网第一、独一无二的

### 「级/极」类
国家级、世界级、顶级、极品、极致、至极、终极、极佳、极好

### 「首/国」类
首创、首款、首家、首发、独一无二、独家、绝无仅有、前无古人、史无前例、中国第一

### 品牌相关
领导品牌、第一品牌、金牌、王牌、驰名商标（不可用于广告宣传）、名牌、著名品牌

## 虚假宣传类

### 功效夸大
- 「立竿见影」「瞬间」「一秒」「即刻」「马上见效」
- 「100%」「保证」「绝对」「肯定」「永不」
- 「包治百病」「治愈率」「有效率达」

### 时间虚假
- 「限时」「最后一天」「仅剩」「最后一批」
- 虚构原价：划线价需有真实成交记录

### 数据夸大
- 「万人疯抢」「卖爆了」「已售100万+」→ 需要真实数据支撑
- 「好评率100%」→ 除非确实无任何差评

## 平台特有违禁词

### 淘宝
- 禁止使用「淘宝」「天猫」「阿里」等平台名暗示官方背书
- 禁止使用「假一赔十」等超出平台规则的承诺

### 拼多多
- 严格禁止价格误导
- 禁止使用「亏本」「清仓」等可能误导消费者的词

### 抖音
- 禁止使用「抖音推荐」「抖音同款」暗示平台背书
- 禁止在商品描述中使用诱导互动话术

## 检测规则

1. **精确匹配**：直接命中违禁词库 → 🔴 违规
2. **变体检测**：谐音/拆字/拼音替代 → 🔴 视为违规
   - 例：「蕞好」「zui好」「最-好」→ 同样违规
3. **语境判断**：部分词在特定语境下可不违规
   - 例：「追求极致品质」→ 🟡 需人工判断（可能擦边）
   - 例：「全网最低价」→ 🔴 明确违规
4. **行业特殊规则**：美妆/食品/保健品额外限制
   - 不得宣传医疗功效
   - 不得使用患者形象
   - 特殊化妆品需标注批准文号

## 安全替换建议

| 违禁表述 | 安全替换 |
|---------|---------|
| 最好/最佳 | 优选、精选、深受喜爱 |
| 第一品牌 | 畅销品牌、口碑品牌 |
| 顶级品质 | 严格品控、高标准的 |
| 最低价 | 性价比之选、实惠价格 |
| 100%有效 | 多数用户反馈良好 |
| 独家 | 本店特色、原创设计 |
| 销量第一 | 累计热销xx件（需真实数据） |
