# 竞品分析方法论

## 竞品识别

### 竞品分层
| 层级 | 定义 | 数量 | 关注重点 |
|------|------|------|----------|
| 直接竞品 | 同品类、同价位、同人群 | 3-5家 | 标题/价格/主图/转化率 |
| 标杆竞品 | 品类TOP3店铺 | 3家 | 整体运营策略、内容风格 |
| 相似竞品 | 不同品类但同人群 | 2-3家 | 人群运营策略、营销方式 |
| 潜在竞品 | 新进入的快速增长型店铺 | 动态 | 差异化打法、新趋势 |

## 核心分析维度

### 1. 商品维度
```
采集字段：
- 标题结构与关键词布局
- 价格（一口价、券后价、SKU价格区间）
- 月销量与累计销量
- 主图风格与卖点文案
- 详情页结构与内容亮点
- SKU数量和规格设计
- 评价总数/好评率/差评关键词
```

### 2. 流量维度
```
分析指标：
- 搜索流量占比 vs 推荐流量占比
- 核心引流关键词（与本店重叠度）
- 付费推广位（直通车/钻展/淘客）
- 内容流量（直播/短视频/图文）
```

### 3. 转化维度
```
分析指标：
- 价格竞争力（本店价格 vs 竞品同款均价）
- 卖点差异化（本店是否有独特卖点）
- 信任度对比（评价数量/DSR/店铺年限）
- 促销力度（满减/赠品/限时折扣）
```

### 4. 服务维度
```
分析指标：
- 客服响应速度
- 发货速度与物流评分
- 售后政策（是否支持7天无理由/运费险）
- 店铺保障（正品保证/破损包赔等）
```

## 分析方法

### SWOT速评法
```
对每个关键竞品做速评：
- Strengths（优势）：做得比我们好的点
- Weaknesses（劣势）：做得比我们差的点
- Opportunities（机会）：我们可切入的差异化空间
- Threats（威胁）：可能抢占我们市场份额的风险
```

### 差异分析矩阵
```
          低价        中价        高价
高颜值  [竞品A]     [竞品B]     [竞品C]
中颜值  [竞品D]     [本店]      [竞品E]
低颜值  [竞品F]     [竞品G]     [竞品H]

目标：找到矩阵中竞争较弱的象限作为切入点
```


## 抓取频率建议

| 数据类型 | 频率 | 说明 |
|---------|------|------|
| 竞品价格/销量 | 每日 | 快速响应价格战 |
| 竞品评价/差评 | 每3日 | 发现竞品弱点 |
| 竞品标题/主图 | 每周 | 检测内容变化 |
| 竞品上新 | 实时 | 第一时间跟进 |
| 竞品整体分析 | 每周 | 周报输出 |
