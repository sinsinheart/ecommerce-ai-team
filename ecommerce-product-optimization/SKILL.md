# 商品智能优化

基于竞品数据分析与AI文案生成，自动化优化店铺商品内容，提升搜索曝光与转化率。

## 触发场景

- 需要优化某个商品的标题、详情页
- 检测到商品流量下滑，需诊断原因
- 上架新品，需生成标题和详情文案
- 平台大促前批量优化商品关键词
- 竞品店铺上了同类新品，需要跟进分析

## 核心工作流

### 1. 竞品数据抓取

```
Task Progress:
- [ ] 确定目标商品和竞品范围（同品类TOP10店铺）
- [ ] 抓取竞品标题、价格、销量、主图、评价关键词
- [ ] 提取竞品高频关键词和差异化卖点
- [ ] 对比本店商品，识别优化空间
```

### 2. 标题与文案AI改写

**标题优化规则：**
- 核心词前置（品类词 + 属性词 + 场景词）
- 包含平台搜索高频词，控制在规定字数内
- 避免堆砌关键词，保持可读性
- 差异化卖点突出（材质/功能/适用人群）

**详情页文案优化规则：**
- 痛点开场 → 产品解决 → 细节展示 → 信任背书 → 行动号召
- 使用场景化描述，而非参数罗列
- 补充实物实拍描述、材质细节说明
- 移动端适配：段落短、重点加粗、图片配文

**主图文案优化规则：**
- 突出核心卖点，8-15字为佳
- 配合促销信息（限时/限量/新品）
- 避免违规字样（最/第一/国家级等）

### 3. 违禁词检测

- 对照平台广告法违禁词库检测
- 检测极限词（最、第一、唯一、顶级等）
- 检测虚假宣传用语
- 检测未授权品牌词/商标词
- 输出检测报告，标注违规位置和建议替换词

### 4. 低流量商品诊断

- 统计近7天/30天商品流量和转化数据
- 对比同品类大盘数据，判断是否异常
- 诊断原因：标题关键词不足、价格不具竞争力、主图点击率低、评价数量少或差评多
- 生成优化建议优先级列表

## 优化建议输出模板

```markdown
## 商品优化报告 — [商品名称]

### 当前问题
- 标题得分：xx/100（关键词覆盖不足）
- 流量趋势：环比下降xx%
- 竞品对比：价格偏高/主图点击率低于同行均值

### 优化方案
1. 标题优化：[新标题建议]
2. 主图优化：[建议方向]
3. 详情优化：[具体建议]

### 预期效果
- 搜索曝光提升：xx%
- 点击率提升：xx%
```
