# 店小二 — 黑客松提交文案

---

## 项目名称

**店小二**

---

## 一句话介绍

店小二——8 位 AI 员工组成的电商全链路运营团队，覆盖拼多多·淘宝·抖音三平台，从选品到口碑治理，7×24 全自动，零代码驱动。

---

## 解决的问题

中小电商商家人手不足、工具碎片化、运营靠感觉。传统方案是买多套工具叠加：客服机器人一笔、ERP一笔、选品插件一笔，还要手动在各平台间切换，数据互相不通。中小商家雇不起完整的运营团队，一个人干选品、客服、发货、售后，24 小时不够用。

---

## 解决方案

基于 EasyClaw 搭建一支 8 人 AI 运营团队——每位 Agent 有明确的岗位职责：

| 角色 | 负责 |
|------|------|
| 🎯 运营总监 | 意图识别、任务分配、多 Agent 协作编排 |
| 🔥 选品专员 | 全网热点扫描、六维评分预测、跨平台选品策略 |
| 🎧 客服主管 | 三平台 7×24 自动接待、差评风险主动拦截 |
| 📦 商品运营 | 竞品分析、AI 标题详情改写、违禁词自动检测 |
| 📊 数据分析师 | 每日三平台日报/周报、异常自动诊断 |
| 🔧 售后专员 | 自动审核退款、5 大恶意订单识别模型 |
| 🛡️ 口碑经理 | 全量差评抓取、AI八类归因、商品/服务/供应链三维优化 |
| 🌐 跨平台分析师 | 三平台统一看板、一键切换、选品对比 |

商家说人话就能驱动团队："帮我看看差评情况""防晒冰袖在哪个平台最好卖""生成今日日报"。

---

## 技术实现

- **11 个 EasyClaw Skill** 模块化设计，通过 orchestrator 协作
- **18 个 Python 脚本**（共 57+ API 封装）对接拼多多/淘宝/抖音三大平台
- **7 项 cron 定时任务**已部署运行（07:00 选品 → 08:00 日报 → 09:00 差评 → 12:00 售后 → 18:00 口碑 → 周一/周三周报）
- **6 大业务闭环**：差评驱动优化、爆款选品上架、运营驾驶舱、售后全链路、健康度巡检、跨平台聚合

---

## 核心差异化

- **从"看到差评"到"根治差评"**：AI 八类归因（尺码/质量/色差/物流/客服/发货/包装/不符）+ 三维优化方案 + 复查追踪闭环
- **从"凭经验选品"到"数据驱动预测"**：六维量化模型（热度/销量/讨论/竞争/空白/季节）+ 跨平台策略推荐
- **从"多工具切换"到"一个团队协作"**：三平台统一看板、一键切换、数据天然互通
- **从"学操作"到"说人话"**：零代码，自然语言驱动全部功能
