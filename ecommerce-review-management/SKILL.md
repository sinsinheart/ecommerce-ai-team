# 差评智能抓取与店铺优化诊断

全链路差评运维机器人，实现差评全量抓取→AI归因→优化建议→复盘预警的自动化闭环，弥补中小商家口碑运营短板。

## 触发场景

- 需要轮询抓取店铺最新差评/中评/追评
- 需要分析差评原因分布，定位店铺核心短板
- 差评率或负面评价超出阈值需要告警
- 需要生成店铺口碑复盘日报/周报
- 需要针对高差评商品输出专项优化方案

## 核心工作流

### 1. 全量差评实时抓取

```
Task Progress:
- [ ] 轮询店铺所有商品评价区、追评区、问答区
- [ ] 筛选1-3星负面评价，自动过滤：
  - [ ] 无效评价（无实质内容、纯符号、无意义灌水）
  - [ ] 水军评价（短时间内大量相似内容）
  - [ ] 重复评价（同一用户对同商品多次相同评价）
- [ ] 收录完整数据：差评内容、下单商品、用户昵称、下单时间、收货地区、差评关键词、配图/视频
- [ ] 同步汇总至店铺口碑数据库
```

### 2. AI差评智能归因分析

**八大归因分类：**
1. **物流延迟** — 发货慢、运输慢、未按时送达
2. **商品质量** — 材质差、做工粗糙、使用后损坏
3. **尺码偏差** — 偏大/偏小、尺码表不准、版型问题
4. **色差问题** — 实物与图片颜色不一致、色差明显
5. **客服态度** — 回复慢、态度差、解决问题不积极
6. **发货速度** — 迟迟不发货、虚假发货
7. **包装破损** — 包装简陋、运输损坏、漏发少发
8. **实物与详情不符** — 功能不符、配件缺失、规格错误

**分析输出：**
- 各类差评占比统计
- 高频负面关键词TOP20
- 差评数量趋势（日/周/月环比）
- 问题TOP5商品排行
- 店铺核心短板定位

### 3. 自动生成店铺专属优化建议

**商品端优化：**
- 尺码不准 → 更新尺码表、增加试穿报告、标注偏大偏小提示
- 色差严重 → 补充自然光实拍图、增加多角度颜色展示
- 质量问题 → 补充材质细节说明、增加质检报告、建议商家升级供应链

**服务端优化：**
- 客服响应慢 → 增加常见问题快捷回复、设置响应时效标准
- 售后态度差 → 优化售后话术、建立补偿机制
- 发货速度慢 → 建议调整发货承诺时效、优化仓库发货流程

**供应链端优化：**
- 物流延迟 → 推荐更换快递渠道方案
- 包装破损 → 升级包装方案（加固/防水/防震）
- 商品质量批次问题 → 标记风险商品，建议下架整改

**风险商品标记：**
- 差评率 >10% → 建议下架整改
- 差评率 5%-10% → 建议优化迭代
- 差评率 <5% 但呈上升趋势 → 关注预警

### 4. 口碑复盘与预警

**日报模板：**
```markdown
# 店铺口碑日报 — YYYY-MM-DD

## 差评概览
- 今日新增差评：xx条 | 差评率：xx% | 环比：+/-xx%
- 累计待处理差评：xx条

## 归因分布
| 类型 | 数量 | 占比 | 环比 |
|------|------|------|------|
| 物流延迟 | | | |
| 商品质量 | | | |
| ... | | | |

## 问题 TOP 商品
## 高频负面关键词
## 优化建议优先级
## 风险预警
```

**预警规则：**
- 店铺日差评率 >5% → 🔴 高风险告警
- 店铺日差评率 >3% → 🟡 关注预警
- 单品日负面评价 >5条 → 单品预警
- DSR评分连续7天下降 → DSR预警
- 同一问题类型连续3天上榜 → 问题升级预警

预警输出自动推送整改清单和优化优先级。
